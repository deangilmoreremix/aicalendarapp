# AI Calendar App

An intelligent, AI-powered calendar and task management application designed to streamline productivity, enhance decision-making, and automate routine workflows through advanced artificial intelligence capabilities.

## 🚀 Overview

This comprehensive application combines traditional calendar functionality with cutting-edge AI features to provide users with intelligent insights, automated task management, and predictive analytics for personal and professional productivity.

- **Task Filtering & Search**: Real-time search with filters by priority, status, assignee, tags, and due dates
- **Subtask Management**: Break down complex tasks into manageable subtasks with individual tracking
- **Task Templates**: Pre-defined task structures for common workflows
- **Bulk Task Operations**: Select and manage multiple tasks simultaneously

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ and npm
- Modern web browser with JavaScript enabled
- Internet connection for AI features and synchronization

### Installation
```bash
# Clone the repository
git clone https://github.com/deangilmoreremix/aicalendarapp.git

# Navigate to the project directory
cd aicalendarapp

# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```

### Configuration

#### Supabase Database Setup
1. Create a Supabase account at [supabase.com](https://supabase.com)
2. Create a new project
3. Go to Settings > API in your Supabase dashboard
4. Copy your Project URL and anon/public key
5. Update the `.env` file in the project root:
   ```env
   VITE_SUPABASE_URL=https://your-project-id.supabase.co
   VITE_SUPABASE_ANON_KEY=your-anon-key-here
   ```
6. Run the SQL migrations in your Supabase SQL editor (found in `supabase/migrations/` if available)

#### Additional Configuration
1. Set up your environment variables for AI services and integrations
2. Configure authentication and authorization settings
3. Configure email and calendar integrations

## 📚 API Documentation

The application provides a comprehensive REST API for integrations:

### Task Management
- `GET /api/tasks` - Retrieve tasks with filtering and pagination
- `POST /api/tasks` - Create new tasks
- `PUT /api/tasks/{id}` - Update existing tasks
- `DELETE /api/tasks/{id}` - Delete tasks

### Contact Management
- `GET /api/contacts` - Retrieve contacts with advanced filtering
- `POST /api/contacts` - Create new contacts
- `PUT /api/contacts/{id}` - Update contact information
- `POST /api/contacts/{id}/enrich` - Trigger AI contact enrichment

### AI Features
- `POST /api/ai/insights` - Generate AI insights for contacts or tasks
- `POST /api/ai/tasks/suggest` - Get AI task suggestions
- `POST /api/ai/meetings/optimize` - Optimize meeting scheduling
